# 🔮 Oráculo Zodíaco (PHP)

## Estrutura do Projeto

- `index.php`: Página inicial com o formulário de entrada.
- `show_zodiac_sign.php`: Processamento da data e exibição da essência do signo.
- `header.php`: Arquivo de cabeçalho compartilhado entre as páginas.
- `signos.xml`: Base de dados dos signos.
- `assets/style.css`: Estilização do projeto.

## Tecnologias

- PHP
- XML
- HTML
- CSS

## Como executar

1. Certifique-se de ter o XAMPP instalado e rodando (Apache).
2. Mova a pasta `Project` para dentro da pasta `htdocs` do seu XAMPP.
3. Abra o navegador e acesse:

http://localhost/Project/index.php
